# Iowa Cities Dashboard

An interactive Shiny dashboard for exploring and comparing Iowa cities across multiple dimensions including demographics, economics, housing, education, crime, healthcare, and environment.

## Features

- **City Rankings**: Compare cities across various metrics
- **City Profiles**: Detailed profiles for each Iowa city  
- **Interactive Maps**: Geographic visualization of city data
- **Trend Analysis**: Historical data trends from 1980-2020
- **Recommendations**: Get personalized city recommendations based on your preferences

## Data Sources

Data compiled from various public sources including US Census, FBI Crime Statistics, and state education records.
