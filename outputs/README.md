# Outputs Directory

This directory contains all generated outputs from the analysis.

## Subdirectories

- **figures/**: Generated plots and visualizations
- **tables/**: Generated tables and summaries
- **reports/**: Rendered R Markdown reports
- **logs/**: Execution logs from `run_all_iowa_analyses.R` (when `--log` flag is used)

## Note

Files in this directory are typically regenerated and should not be manually edited.
Add output files to `.gitignore` if they are large or frequently regenerated.
